O que são Clients?

Um client é um software que solicita serviços de um server. Normalmente usamos browsers como clients para acessar páginas web.

Voltar ao índice

O que são Servers?

Um server processa pedidos de clients e devolve respostas. Pode armazenar dados, hospedar sites e fornecer serviços de rede.

Como é uma estrutura básica de um HTML?

Um HTML básico contém DOCTYPE, <html>, <head> e <body>. Dentro do body, podemos adicionar textos, imagens, listas e links.

Exemplos de novas tags

O atributo HTML é usado para estruturar páginas.

Podemos usar texto antigo para mostrar conteúdo removido e texto em vermelho para destacar algo.

Também podemos adicionar subscrito ou sobrescrito em fórmulas, como H2O ou E = mc2.

HTML é a linguagem padrão para criar páginas na web.
